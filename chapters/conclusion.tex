\label{sec:conclusion}
\hspace*{0.6cm} Энэхүү бакалаврын судалгааны ажлын хүрээнд МУИС-ийн Дипломын ажлын удирдлагын системийн хэрэглэгчийн интерфэйс дээр React болон JSF технологийг хослуулсан холимог микрофронтэнд архитектурыг туршиж, ISO/IEC 25010 стандарт болон DORA, ATAM аргачлалаар үнэлж дүгнэлээ. Дэвшүүлсэн гурван зорилт бүрэн биелэгдэв:

\begin{itemize}
    \item Судалгаа хийх зорилтоор микрофронтэнд архитектурын онол, арга зүй болон ISO/IEC 25010 стандартыг судалж, холбогдох 10 эрдэм шинжилгээний бүтээлийн тойм хийв.
    \item Туршилтын загвар боловсруулах зорилтоор Vite Module Federation ашиглан JSF хост болон React MFE модулиудыг нэгтгэсэн туршилтын загварыг боловсруулж, уг туршилтын загварын дагуу МУИС-ын Дипломын ажлын удирдлагын системийг хэрэгжүүлэв.
    \item Туршилт ба үнэлгээ хийх зорилтоор ISO/IEC 25010 стандартын 7 чанарын үзүүлэлтийг Google Lighthouse, SonarQube, madge, Jest, Cypress хэрэгслүүдээр хэмжиж, JSF болон React MFE архитектурыг харьцуулан шинжилж, ATAM аргачлалаар харьцуулалт хийв.
\end{itemize}
